## 🔹 About

Desenvolvedor Full Stack em formação (Análise e Desenvolvimento de Sistemas — UCB, conclusão prevista para o final de 2026), atuando profissionalmente como **Desenvolvedor Full Stack Trainee** na Soluction Eventos, com preferência declarada por **back-end**.

No dia a dia, construo e mantenho um sistema interno de gestão real — produção, compras, financeiro, contratos, ordem de serviço e auditoria — usando **NestJS**, **Next.js**, **TypeScript** e **PostgreSQL/TypeORM**. Em paralelo, desenvolvo projetos pessoais em **Java/Spring Boot** para consolidar arquitetura em camadas, DTOs, persistência com JPA/Hibernate e boas práticas de API REST.

Tenho foco em código limpo, modelagem de dados consistente e entrega com testes (Jest, JUnit), participando ativamente de sprints, code review e pipelines de CI/CD no dia a dia de trabalho.
